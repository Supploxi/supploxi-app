Store Integrations

Connect Supploxi to your e-commerce store to sync products and inventory automatically. Follow the step-by-step guide for your platform.

Shopify

Most Popular

Sync your Supploxi inventory with your Shopify store. Track stock levels, get notified when products run low, and keep everything in sync.

Step-by-step setup guide

1
Go to your Shopify Admin

Log into your Shopify store and navigate to Apps → App and sales channel settings

Open Shopify Admin ↗
2
Create a Private App / Custom App

Go to Settings → Apps and sales channels → Develop apps. Create a new app called "Supploxi Integration".

3
Set API Permissions

In API credentials → Configure Admin API scopes, enable:

read_products write_products read_inventory write_inventory read_orders
4
Get your API credentials

After installing the app, copy your Admin API access token and your store URL (e.g. mystore.myshopify.com)

5
Enter your credentials below

WooCommerce

WordPress

Connect your WooCommerce store via REST API. Sync stock levels and get reorder alerts when your WordPress inventory runs low.

Step-by-step setup guide

1
Go to WooCommerce → Settings → Advanced

In your WordPress admin, navigate to WooCommerce → Settings → Advanced → REST API

2
Create API Key

Click Add Key. Set description to "Supploxi", user to your admin account, and permissions to Read/Write.

3
Copy Consumer Key and Consumer Secret

Important: These keys are shown only once. Copy them immediately after creation.

4
Enter your credentials

BigCommerce

Integrate with your BigCommerce store to monitor inventory levels and coordinate supplier orders with your online store.

Step-by-step setup guide

1
Go to Advanced Settings → API Accounts

In your BigCommerce control panel, go to Settings → API → API Accounts → Create API Account → V2/V3 API Token

2
Set permissions

Enable Products: Modify and Store Inventory: Modify permissions.

3
Enter your credentials
Found in your store URL: store-HASH.mybigcommerce.com
